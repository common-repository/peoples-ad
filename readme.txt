Displays Peoples-Ad Iframe after payment confirmation in thank you page

== Description ==

This is an official plugin for the Peoples-Ad marketing platform.
The service provides ecommerce stores the ability to promote their products by using their customers as advocates.

The plugin uses Peoples-Ad as a third party service to display an invite to the user to start sharing an advert to their friends. The invite is displayed on the confirmation page, and it will only display content if an advert has been set up for the corresponing product. 
If the customer wants to use Peoples-Ad, they will be directed to the Peoples-Ad website where they will complete the sharing. The customer will then earn refferral commission for every unique click.

An advert can be set up on the Peoples-Ad portal and linked to your product using the product id as set up on your webstore.

Steps to set up Iframe:

1. Open menu "Peoples-Ad" from dashboard
2. Get the Store Id from [Peoples-Ad](https://www.peoples-ad.com/Developer/WooCommerce)
3. Save the settings

== Installation ==

= Manual Installation =

1. Upload the plugin files to the `/wp-content/plugins/` directory, 
2. Unzip the file
3. Activate the plugin through the 'Plugins' screen in WordPress4.
